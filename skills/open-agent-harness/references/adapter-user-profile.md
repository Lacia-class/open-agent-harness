# User Profile Adapter

Use this adapter for optional collaboration preferences.

The public core provides interaction principles. Personal cognitive models, long-term memory, private preferences, or individual-specific response rules belong in this adapter or a private pack.

## Capability Contract

```text
preferred_language:
verbosity:
confirmation_style:
risk_gate_style:
fatigue_signals:
format_preferences:
forbidden_assumptions:
memory_policy:
```

## Privacy Rules

- Do not publish a real person's profile without explicit consent.
- Treat psychological inferences as private by default.
- Store observations as revisable preferences, not fixed identity claims.
- Public examples must use synthetic profiles.
- Long-term memory writes require explicit user consent.
